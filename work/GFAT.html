Project Description:

This project was another one of my primary efforts during the Summer 2016 Data Visualization Internship at the NASA Jet Propulsion Lab + Caltech. I worked with fellow interns Matt Edwards and Xiangjun Shi-Trofimov. to create a visualization solution to explore some cutting edge research questions the Mark Simons lab has about interferomtery based models of glacier flow. In order to develop better models for simulating glacier flow and to improve data collection techniques, researchers in the Simons lab wanted an integrated solution for visualizing information about the complex intereferometry data input into the flow model, and the output of that model. This flow model can have a multitude of components operating on vastly different time scales, so we had to create a visual grammar that would enable ea user to look for potentially interesting flow characteristics or defects in the model across multiple scales.
