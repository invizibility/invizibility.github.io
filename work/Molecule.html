Project Description:

This project is another instance of me looking to piece together creative workflows that involve tools that are traditionally used in a scientific context repurposed to an artistic end. I have been interested in the Molecular Maya plugin for some time now, which allows users to easily import complex molecular geometries into Maya for rendering of scientific art. I wanted to abstract away the underlying complexity and symmetry of a molecule and replace each instance of a particular molecule with some mechanical component, to create a form with underlying organic components but with an outwardly mechanical appearance. The video shows a rendering of a particular nucleocapsid protein expressed in E. Coli with all its molecules replaced with pink Cadillac Escalades. This is my first foray into this territory and I intend to utilize this workflow I have discovered to piece to create many more of these forms in proper design study fashion.
