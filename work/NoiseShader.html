2012
Project Description:

This project was another assignment I completed for Meg Mitchell's Art 448 "Web and Interactive Media" class alongside the music library visualization. On this particular assignment, we were asked to create dynamic visual compositions with Processing. I was interested in exploring 3 main motifs in this composition: Perlin noise, particle systems, and rotational symmetry. I created a Processing sketch, using vector field code from this project, that brings these elements together with several interactive functionalities including: adding particles, rotating the particle system, deforming the particle path, and modulating the degrees of rotational symmetry of the system. I passed the modulated noise data through a version of Processing's Gaussian blur shader that I modified to have several time varying characteristics.
