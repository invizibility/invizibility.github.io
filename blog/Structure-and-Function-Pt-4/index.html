Structure and Function Pt. 4: Diving into Feature Design

Armed with an understanding of both the structure data ontology and function data ontology relevant to our visualization question, in conjunction the list of ideal interface characteristics mentioned in Pt 3, I began to dive into the feature design problem. If we were to develop an experiment selection interface for our data of interest (Rao et. al cell lines + related CHiP-seq experiments), what would this interface look like and how would it guide the user? Furthermore, what types of visual grammar explored in the background survey research could be leveraged to make clear relationships between heterogeneous genomics data types? The end goal of this step was to yield a prototype we could use to intitiate our iterative user testing process.
