Background & Prior Work

2016-09-18

There are several different types of media that need to be explored to consolidate a history of prior work in genomics visualization.

  • media consisting of visualizations that describe a problem, process, or observation relating to genomic data.
  • media describing the problem space of visualization in genomics. this has to do with genomics’ place within the epistemological regime of systems biology
  • media that attempts to create criteria for good/bad visualizations

In this post I will tackle the first area. I’ve tried to select visualizations that I think are either effective or ineffective for notable reasons; there are a lot of bad visualizations out there so it is important to learn from ‘near misses’ instead of trying to extract meaning from something that is a complete failure.

This blog post by Mo Rahman, the founder of GeneDrop, provides a good overview of the most common types of visual grammar used in contemporary genomics problem solving. Going through each of the 5 categories outlined in his post, I’ve aggregated some visualization strategies I think are interesting.

Track Visualization

The UCSC genome browser is considered the standard of this type of visualization. This project began to showcase the initial findings of the Human Genome Project, and continues to be a widely used repository for genomic alignment and annotation data for several different species.
